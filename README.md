\# Church CMS



Full-stack Church Management System built with Spring Boot and React.



\## 🚀 Tech Stack

\- \*\*Backend:\*\* Java, Spring Boot, Spring Data JPA, MySQL

\- \*\*Frontend:\*\* React, Vite, Axios

\- \*\*Tools:\*\* Git, GitHub, Maven
